# Sistema de Notas Detalhado

Este projeto consiste na criação de um sistema em Python para registrar as notas de 25 alunos, calcular a média geral da turma, identificar a maior e a menor nota, e classificar os alunos em diferentes categorias de desempenho. O sistema também contará com a contagem de alunos em cada categoria.

## Objetivo

O objetivo deste trabalho é implementar um sistema simples de avaliação que ajude a organizar e analisar o desempenho dos alunos, além de permitir que os professores possam ver facilmente a distribuição das notas e o número de alunos em cada faixa de desempenho.

## Tecnologias Utilizadas

- Python 3.x
- Listas e Funções
